# CheckboxGroup 复选框组

```JS
import Vue from 'vue'
import { CheckboxGroup } from 'yum'

Vue.use(CheckboxGroup)
```

## Props

| 属性 | 描述 | 类型 | 可选值 | 默认值 | 版本支持 |
| - | - | - | - | - | - |
| v-model | 当前的值 | Array | - | - | - |
| min | 最少可选个数 | Number | - | - | - |
| max | 最多可选个数 | Number | - | - | - |

## Slots

| 名字 | 说明 |
| - | - |
